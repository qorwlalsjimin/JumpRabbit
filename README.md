# JumpRabbit
2022년에 3개월동안 진행한 JAVA 프로젝트입니다.<br>
`백지민`과 `배서연` 둘이서 개발했으며 뉴미디어디자인과 `이현정`의 디자인입니다<br><br>

* 점프, 더블점프
